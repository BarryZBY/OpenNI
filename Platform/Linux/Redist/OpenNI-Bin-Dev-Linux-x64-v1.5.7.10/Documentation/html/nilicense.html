niLicense

niLicense allows adding or removing licenses from the global license registry. This utility is commonly used by modules installations.

Usage

niLicense [options] <vendor> <key>

The following options can be used:

  • -r Registers a license to the global registry.
  • -u Unregisters a module from the global registry.

If no option is supplied, -r is assumed.
